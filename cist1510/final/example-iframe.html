iFrames

iFrames allow the display of other pages both internal and external. Alongside the standard, iFrames support the following attributes in HTML5:

Name Description
name Specifies a name for the iframe.
sandbox Sets restrictions for content in an iframe.
src Specifies the source URL.
srcdoc Sets HTML content of the page to display in an iframe.
width Sets the width of an iframe.

While managing iframes, note that some websites do not permit the hotlinking of them due to security reasons. Generally, this can be fixed using a local proxy, however may not be efficient or properly handle all requests.

Iframe of this page via the following HTML:

<iframe width="400" height="200" src="https://www.gwinnetttech.edu" allowfullscreen></iframe>
