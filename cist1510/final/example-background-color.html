background-color

The background-color CSS Property

The background-color CSS property can be applied to change the color behind content. It accepts HEX, RGB, RGBA, HSL, and HSLA values, standard color names, and the following:

Name Description
transparent Sets the background transparent.
initial Sets the property to its default value based on the browser.
inherit Uses the background-color property of the container's parent element.
I am content!

The background color of the code block above was changed by specifying CSS for it as follows:

background-color: #5c99ed;

The full HTML and CSS mix for the example is:

<code style="background-color: #5c99ed;">I am content!</code>

It can also be placed in an external CSS stylesheet or in a HTML page's style tags:

<style> code { background-color: #5c99ed; } </style> <code>I am content!</code>
